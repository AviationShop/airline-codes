# Airline Codes (IATA & ICAO)

Open dataset of **86 major world airlines** with their **IATA** (2-char) and **ICAO** (3-char) codes, airline name and country — e.g. `AA` / `AAL` American Airlines, `LH` / `DLH` Lufthansa, `EK` / `UAE` Emirates.

JSON and CSV, for flight trackers, booking tools, plane-spotting apps and aviation projects.

## Files
- [`airline-codes.json`](airline-codes.json) · [`airline-codes.csv`](airline-codes.csv)

## Example
```json
{ "iata": "BA", "icao": "BAW", "airline": "British Airways", "country": "United Kingdom" }
```

This list covers the world's major scheduled carriers. Contributions to extend coverage are welcome.
